### Resumen:
El proyecto desarrollado es una API REST para el sistema de comisiones para una cadena hotelera. Se desarrollo la REST API
basada en el sistema existente, en este caso se trabajo en el backend desarrollando todos los endpoints del sistema.
Dentro del proyecto se manejaron diferentes tecnologías para la comunicación del equipo y el desarrollo del proyecto, SCRUM 
fue la metodología empleada.

### Tecnologías

* VPN GlobalProtect
* Jira Software
* Tecnologías PHP, JSON, JSON Web Tokens
* Yii2, Framework para PHP
* Microsoft Teams
