### Resumen:
Se desarrollo una plataforma de casino online que utiliza criptomonedas. Este casino cuenta con diferentes juegos desarrollados algoritmicamente para ser 100% Provably Fair.
Los usuarios pueden enviar y recibir criptomonedas las cuales se agregan al balance de sus cuentas en forma de "chips" las cuales se utilizan para poder apostar.
El sistema cuenta con varios modulos, así como con un panel de administración donde se pueden visualizar analiticas y editar configuraciones.

* Aplicación desarrollada con el stack MERN
* Implementación con proveedores de juegos de slots
