### Resumen:
El sistema web desarrollado tiene el fin de mejorar el seguimiento de siniestros de una aseguradora de autos. Mediante este sistema el operador
puede registrar un nuevo siniestro y comenzar a dar seguimiento del estado del mismo. Puede administrar la documentación del siniestro y de los usuarios
asegurados, revisar los pagos, servicios, estimaciones, etc.


### Tecnologías

* ReactJS
* PHP
* MongoDB
* JQuery
* Bootstrap
